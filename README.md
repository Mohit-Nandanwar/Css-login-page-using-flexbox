# Split-Screen Login Page

This project demonstrates a simple split-screen web layout using **HTML** and **CSS**.  
The left half of the screen displays an image, while the right half contains a login form with email and password fields, a login button, and a "Forgot Password?" link.

## 🚀 Features
- **Responsive split layout**: Left side for image, right side for login form.
- **Login form elements**:
  - Email input field
  - Password input field
  - Login button
  - "Forgot Password?" link
- **Clean design** with hover effects on the login button and link.
